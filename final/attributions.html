Attributions

Credits for resources used in this project.

CapitalFlow was built as an individual academic project for WDD231. The following external resources were used in its development.

Icons

  • All icons are from Bootstrap Icons — MIT License, by The Bootstrap Authors.

Typography

  • IBM Plex Sans — Open Font License, by IBM. Served via Google Fonts.

APIs & Data

  • ExchangeRate-API — Real-time and historical currency exchange rates. Free tier used.
  • Frankfurter API — Open-source exchange rate API based on European Central Bank data.

Financial Tips Content

  • All financial tips were written originally for this project, inspired by general personal finance principles from sources including Investopedia and NerdWallet.

Tools & Technologies

  • HTML5, CSS3, and JavaScript (ES Modules) — core web technologies, no frameworks used.
  • Canvas API — native browser API used for chart rendering.
  • localStorage API — native browser API used for client-side data persistence.
  • GitHub Pages — free static site hosting.
  • Visual Studio Code — code editor.

Validation & Testing

  • W3C HTML Validator — HTML validation.
  • W3C CSS Validator — CSS validation.
  • WAVE Accessibility Tool — accessibility evaluation.
